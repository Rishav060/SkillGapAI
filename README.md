# SkillGapAI

SkillGapAI is a web-based application that analyzes the gap between a user's resume and a job description.

It helps users understand what skills they already have, what they are missing, and provides a clear roadmap to become job-ready.

---

## 🚀 Features

* Extracts skills from resume and job description
* Identifies matched and missing skills
* Calculates job readiness score
* Generates a personalized learning roadmap
* Handles edge cases like zero skill overlap

---

## 🧠 How It Works

* Uses a predefined dataset of technical skills
* Matches skills using string comparison in JavaScript
* Calculates:

  * Matched skills (intersection)
  * Missing skills (difference)
* Generates a roadmap based on missing skills

---

## 🤖 Use of AI

AI tools were used during development to assist with UI design and code structuring.

However, the core logic — including skill extraction, matching, and roadmap generation — was understood, tested, and refined manually to ensure correct and reliable behavior.
